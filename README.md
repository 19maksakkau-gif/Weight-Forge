Weight Forge — Конвертер весов на 7 языках
Weight Forge — это коллекция из семи независимых реализаций конвертера единиц массы (веса). Каждая версия работает на своём языке программирования и поддерживает полный набор метрических, имперских и специальных единиц. Проект демонстрирует кроссплатформенный подход и предоставляет удобный инструмент для преобразования весовых величин с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
⚖️ Поддержка множества единиц:

Метрические: миллиграмм (mg), грамм (g), килограмм (kg), центнер (cwt), тонна (t)

Имперские: унция (oz), фунт (lb), стоун (st), американская тонна (us_t)

Специальные: карат (ct), гран (gr), атомная единица массы (amu)

🔄 Гибкое преобразование между любыми двумя единицами

📂 Пакетный режим – конвертация списка значений из файла (по одному на строку)

📊 Таблица эквивалентов – генерация диапазона значений для визуализации

💾 Экспорт результатов в CSV, TXT или JSON

🖥️ Интерфейсы:

Командная строка (CLI) с гибкими опциями

Графический интерфейс (GUI) в Python, Java и C#

Веб-интерфейс в PHP (и браузерный JavaScript)

🧮 Высокая точность (до 10 знаков после запятой)

⚡ Быстрая обработка больших массивов данных

Использование (общий синтаксис)
CLI (все версии):

bash
weight_convert --value 100 --from kg --to lb --precision 3
Примеры:

Конвертировать 5 фунтов в килограммы:

bash
go run weight_convert.go -value 5 -from lb -to kg
Сгенерировать таблицу от 0 до 10 кг с шагом 0.5:

bash
node weight_convert.js --range 0,10,0.5 --from kg --to oz
Пакетная обработка из файла values.txt:

bash
php weight_convert.php --batch values.txt --from g --to lb --output results.csv
